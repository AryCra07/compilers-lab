# Lex Parser

## C++ version

- put the code you wanna parse in `data/in`
- put your file to storage results in `data/out`

### Run

```bash
chmod +x build.sh
./build.sh
cd build
./LexParser [in.txt] [out.txt]
```

## Lex version

- put the code you wanna parse in `in.txt`

### Run

```bash
chmod +x build.sh
./build.sh
./LexParser [in.txt](optional)
```
